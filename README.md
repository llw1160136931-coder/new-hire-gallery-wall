# 新员工展示墙系统

一个 Vite + React 前端项目，用于设计新员工培训作品展示墙的页面和交互原型。

## 运行方式

安装依赖：

```powershell
npm install
```

启动开发服务：

```powershell
npm run dev
```

## 功能

- 学员发布培训作品或 AI 作品，支持上传图片、填写图片地址和作品链接。
- 新作品默认进入待审核状态。
- 管理员可在审核中心通过或打回作品。
- 通过审核的作品展示在作品墙，支持点赞和投票。
- 首页课程表自动区分已结束、进行中、未开始。
- 个人中心支持编辑姓名、头像、毕业院校、MBTI、星座和性别。

当前版本先聚焦前端页面和视觉布局，后续可以接入真实接口、上传、审核流和数据库。
